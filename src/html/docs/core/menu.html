Class Description Type
.Menu Set Menu to use flexbox and aligns items to the left by default Component
.Menu[Xs,Sm,Md,Lg] Used in conjunction with Menu--vertical to break to horizonal at 480px, 640px, 768px or 960px Responsive modifier
.Menu--right Aligns items to the right Component modifier
.Menu--center Aligns items centrally Component modifier
.Menu--expanded Menu items expand to fill the available space Component modifier
.Menu--vertical Aligned menu items vertically in a stacked menu Component modifier
.Menu--flush Removes the left margin of a nested vertical menu Component modifier
.Menu-item A menu item Sub-element
.Menu-item--right Aligns menu item to the right if the menu is left aligned Sub-element modifier
.Menu-item--left Aligns menu item to the left if the menu is right aligned Sub-element modifier
.Menu-item.is-active A active menu item Active state
.Menu-link A menu link Sub-element
.Menu-title A menu title Sub-element
