Javascript: Button module

You can toggle states via an asynchronously loaded javascript module. Just add the data attribute data-module="Button"

<button class="Button" data-module="Button">Toggle me</button>
<button class="Button Button--primary" data-module="Button">Toggle me</button>
<button class="Button Button--hover" data-module="Button">Toggle me</button>

Loading submit button

The button module can also be used with the '.Button--loading' component modifier to create a button that, when clicked shows a loading spinner icon. After this the button can not be toggled back.
